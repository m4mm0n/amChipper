# Commercial Licensing

The public repository is GPL-3.0-only.

GPLv3 is not a non-commercial license. It allows commercial distribution of the GPL-covered version, but redistribution must preserve the GPL freedoms and source availability.

For Geir Gustavsen / ZeroLinez Softworx to release proprietary or commercial editions later, the project owner must retain relicensing rights. This repository does that by requiring contributors to agree to `CONTRIBUTOR_TERMS.md`.

Before accepting outside contributors or shipping a commercial edition, have the contributor terms reviewed by a lawyer.
